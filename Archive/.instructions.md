## Error Handling

### Logging
```python
import logging

logger = logging.getLogger(__name__)

try:
    result = process_data(input_path)
    logger.info(f"Successfully processed {len(result)} records")
except FileNotFoundError as e:
    logger.error(f"Input file not found: {e}")
    raise
except Exception as e:
    logger.error(f"Unexpected error: {e}", exc_info=True)
    raise
```

### Best Practices
- Use specific exception types
- Log with appropriate levels (DEBUG, INFO, WARNING, ERROR)
- Include context in error messages
- Clean up resources in finally blocks
- Don't use bare `except:` clauses

## File Operations

### Python
```python
from pathlib import Path

# Preferred
input_path = Path("data/input.csv")
if input_path.exists():
    with input_path.open() as f:
        data = f.read()

# Avoid
import os
if os.path.exists("data/input.csv"):
    f = open("data/input.csv")  # No context manager
    data = f.read()
    f.close()
```

### Best Practices
- Use absolute paths in file operations
- Handle missing files gracefully
- Use context managers for file I/O
- Close resources properly
- Validate file formats
